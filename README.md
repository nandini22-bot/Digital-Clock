# ⏰ Digital Clock with Alarm System

A simple and interactive digital clock application with alarm functionality, designed with a clean UI and real-time updates.

## 🚀 Features

* Real-time digital clock display
* Alarm setting functionality
* Persistent alarm storage using JSON
* Lightweight and responsive interface

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
* PHP

## 📂 Project Structure

index.html
style.css
script.js
react-app.js
save_alarm.php
alarms.json
